What is CSS?

  1. CSS stands for Cascading Style Sheet .
  2. CSS describe how HTML elements are to be displayed on the screen, paper or in other media.
  3. CSS is not necessary CSS saves a lot of work. It can control the layout web pages all at once
  4. External stylesheet are stored in CSS files.

CSS can be added to HTML documents in three ways and these are :

  • Inline:- by using style attribute inside the element
  • Internal:- by using a style elememt in the head section
  • External:- by using a link element to link an external CSS file. However, in this tutorial we will use inline and internal styles,because this is easier to demenstrate and easier for you to use to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple element.ID, on other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directily to an element and it's used by JavaScrift.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of style can be applied to the element.

There is awide range of selectors available. Two of the most commonly used are class and Id. Both are used target an elements to which a style should be applied.
